### 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/Winuel/winuel.git
cd winuel

# 检出版本
git checkout develop
git pull origin develop

# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build
```

### ⚠️ Beta版本说明

此为Beta测试版本，不建议用于生产环境。主要用途：
- 功能测试和验证
- 收集用户反馈
- 发现和修复问题
- 准备正式版本
