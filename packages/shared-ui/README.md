# @winuel/shared-ui

Winuel 共享 UI 组件包

## 安装

```bash
npm install @winuel/shared-ui
# 或
pnpm add @winuel/shared-ui
```

## 使用

```typescript
import { SomeComponent } from '@winuel/shared-ui'
```

## 特性

- Vue 3 组件库
- TypeScript 支持
- 响应式设计
- Tailwind CSS 样式

## 依赖

- @winuel/shared-core
- vue
- vue-router
