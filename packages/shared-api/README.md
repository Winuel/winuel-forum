# @winuel/shared-api

Winuel 共享 API 客户端包

## 安装

```bash
npm install @winuel/shared-api
# 或
pnpm add @winuel/shared-api
```

## 使用

```typescript
import { createApiClient } from '@winuel/shared-api'
import { client } from '@winuel/shared-api/client'
```

## 特性

- 类型安全的 API 客户端
- 基于 Axios 的请求封装
- 统一的错误处理
- 请求/响应拦截器

## 依赖

- @winuel/shared-core
- axios
