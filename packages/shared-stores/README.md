# @winuel/shared-stores

Winuel 共享状态管理包

## 安装

```bash
npm install @winuel/shared-stores
# 或
pnpm add @winuel/shared-stores
```

## 使用

```typescript
import { useUserStore } from '@winuel/shared-stores'
import { userStore } from '@winuel/shared-stores/user'
```

## 特性

- Pinia 状态管理
- 用户状态管理
- UI 状态管理
- 持久化存储支持

## 依赖

- @winuel/shared-core
- pinia
- vue
